P.A.R.T.S: PC Assembly and Rig Tracking System

PARTS is a program that allows you to browse, manage, and organize computer hardware components for building PCs.

The program shall use datasets of computer hardware from Kaggle. (https://www.kaggle.com/datasets/dilshaansandhu/general-computer-hardware-dataset)

    Core Program Functionalities
        * View parts
            - Filter parts by specific query
        * Create build
        * Save build
        * Load build
        * Delete build

    Part Management
        * Add part to build
        * Remove part from build
    
    Logic
        * Compatibility checking
        * Price calculation
        * Wattage calculation + 100W
